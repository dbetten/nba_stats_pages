Model Notes

The viewer uses one canonical calibrated projection model. Season-level college rows in All Stats are display context only; NBA projections remain player-level draft evaluations. Visible binary probabilities are coherence-adjusted so Starter, STAR+, and SUPERSTAR cannot exceed the non-OUT probability; raw independent scores remain in the report files for audits. Projection-year rows with incomplete required features are marked as UNSCORED_MISSING_DATA.
